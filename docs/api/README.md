# API / Contracts 说明

本目录用于管理 NanoPaste 的接口文档与契约说明，服务于桌面端与 Mock Server 的一致性开发。

## 目录职责

- 记录接口定义约束。
- 约定请求/响应数据结构。
- 作为 `packages/contracts` 的文档索引入口。

## 当前状态

- 已落地 `packages/contracts/v1` 契约定义（仅定义，不含运行逻辑）。
